## 💡 Java Fresher ke liye TIPS

### ✅ Apply karne se pehle yeh cheezein taiyaar karein:
- **Resume** mein Java Projects zaroor likho (College ya Personal)
- **GitHub Profile** banao aur projects upload karo
- **Core Java + Spring Boot** ka basic knowledge hona chahiye
- **SQL/MySQL** basics bhi jaanein

### 📚 Top Skills Jo Companies Dhundti Hain (Fresher Level):
```
✔ Core Java (OOP, Collections, Exception Handling)
✔ Spring / Spring Boot (basics)
✔ MySQL / JDBC
✔ Git & GitHub
✔ REST API basics
✔ Data Structures & Algorithms
```
